ガチャシミュレーター


ソーシャルゲームなどで一般的に見られる「ガチャ」機能をシミュレートするためのWebアプリケーションです。プレイヤーとしてガチャを引く体験ができるだけでなく、管理者としてガチャの排出キャラクターや確率を自由に設計・シミュレーションすることもできます。


事前準備
以下のコードをそれぞれフロントエンド側のターミナルとバックエンド側のターミナルで、一行ずつ実行してください。

・フロントエンド

　1. ターミナル上でフロントエンドディレクトリに移動する。
　　　cd {保存場所}/UTF8/app/frontend

　2. 必要なパッケージをインストールする。
　　　yarn install

　3. フロントエンドのサーバを起動する。
　　　yarn dev


・バックエンド

　1. ターミナル上でバックエンドディレクトリに移動する。
　　　cd {保存場所}/UTF8/app/backend

　2. Pythonの仮想環境を作成・有効化
　　　python -m venv venv
　　　source venv/bin/activate  # Mac/Linuxの場合
　　　venvScriptsactivate    # Windowsの場合

　3. 必要なライブラリをインストール
　　　pip install -r requirements.txt

　4. データベースの初期化とサンプルデータの投入
　　　chmod +x initialize_db.sh
　　　./initialize_db.sh

　5. バックエンドサーバーを起動
　　　python manage.py runserver --settings=config.settings.development


アプリケーションの起動

サーバの起動後はWeb上のローカルホストで、フロントエンドとバックエンドにアクセスすることができます。

　フロントエンド：http://localhost:3000

　バックエンド：http://localhost:8000


使い方

http://localhost:3000/main/loginにアクセスするか、ヘッダーのログアウトボタンを押すことでログイン画面に移行します。
initialize_db.sh によって作成された以下の初期アカウントでログインすることで、アプリケーションを操作することができます。

・管理者アカウント

　　ユーザー名： admin

　　パスワード： adminpass

・一般ユーザーアカウント

　　ユーザー名： player

　　パスワード： playerpass


主な機能
・プレイヤー向け機能

　・ガチャ実行： 1回または10回連続でガチャを引くことができます。

　・キャラクター一覧： ガチャで手に入れたキャラクターの確認ができます。

　・ガチャ履歴： いつ、どのガチャから何が出たかの履歴を閲覧できます。

　・ユーザー認証： ユーザーごとのデータ管理が可能です。

・管理者向け機能

　・ガチャ管理： ガチャの作成、編集、削除ができます。

　・キャラクター管理： 排出対象となるキャラクターの作成、編集、削除ができます。

　・排出率設定： ガチャごとにレアリティ別の排出率や、特定キャラクターのピックアップ設定などを細かく定義できます。
